nodejs部分实用第三方模块介绍

mime（获取文件类型）

作用：

在设置响应报文时自动获取响应体文件的文件类型

使用方法：

  1. npm install mime
  2. const mime = require('mime')
  3. mime.getType(传入需要类型的文件名或文件路径)

特点：

  1. 会自动根据文件扩展名得到mime类型(content-type)
  2. 也可以根据文件路径得到mime类型
