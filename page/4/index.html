let&const

let & const

一、 let

特点：

  1. 不会预解析

  2. 不允许重复声明

  3. 块级作用域

二、 const

特点：

  1. 不允许修改值
  2. 定义的时候必须初始化(赋值)👆
  3. const声明的变量是对象时代表其指针不可改变👈(可以改对象内容)
  4. let有的特点const也有

一般变量不会再修改值了就用 const ，并且可以在改变值的时候思考为什么修改(有点像标签的 id )
只要是复杂类型一般都用 const 定义(内容可以修改，但是数据类型不能变)(包括数组，对象，函数)
