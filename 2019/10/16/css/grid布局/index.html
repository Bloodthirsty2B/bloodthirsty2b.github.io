grid布局语法参考

将网页划分成一个个网格，可以任意组合不同的网格

grid布局语法参考

grid布局是将网页划分成一个个网格，并将其任意组合的布局方法。类似于以前的table布局。

  • 和flex布局的区别：flex是围绕轴线进行的一系列布局方法；而grid是将容器划分为行和列进行布局，其灵活性更高。
  • 兼容性：有较大兼容性问题，参考caniuse-grid

容器属性

布局属性：

  1. display: grid; 块级元素
  2. display: inline-grid; 行内块元素

设置属性：

  1. grid-template-columns

  2. grid-template-rows:; => 语法和 grid-template-columns 相同，对容器分行；

  1. grid-column-gap: 20px; => 给每列之间设置20px的间距 （grid-row-gap: ; 用法相同）；
  1. grid-gap: <grid-row-gap> <grid-column-gap>; => 将grid-column-gap和grid-row-gap 合写。如果省略了第二个值，那么默认第二个值和第一个相同；

    根据最新标准，上面三个属性名的grid-前缀已经删除，grid-column-gap和grid-row-gap写成column-gap和row-gap，grid-gap写成gap。

  1. grid-template-areas

注意，区域的命名会影响到网格线。每个区域的起始网格线，会自动命名为区域名-start，终止网格线自动命名为区域名-end。

  1. grid-auto-flow: column; => 将先行后列的排列顺序改为先列后行的顺序
  1. grid-auto-flow: row dense; => 先行后列，并且尽可能紧密填满，尽量不出现空格。

  1. justify-items、align-items、place-items

  2. justify-content、align-content、place-content

  3. grid-auto-rows

项目属性

  1. grid-column-start、grid-column-end、grid-row-start、grid-row-end、grid-column、grid-row

  2. grid-area: d => 指定项目放在哪一个区域。

  1. justify-self、align-self、place-self
